# Annual reports (PDFs)

The Poverty 360 timeline download buttons expect these files in this folder:

- `2019-annual-report.pdf`
- `2020-annual-report.pdf`
- `2021-annual-report.pdf`

Add the PDFs here and commit them to the same branch that deploys [Poverty 360 on GitHub Pages](https://s-abdul-ai.github.io/poverty360/). Until the files exist, those links will return 404.

If the reports are large, consider hosting them on Google Drive or Dropbox and changing the `href` in `index.html` to the public download URLs instead.
